# MPSim

*MPSim - программа написанная на C# для понимания логики работы многофозного конвейера.*

### Стек

- **Язык:** C# (.NET 8)
- **Интерфейс:** WPF + Material Design Colors/Themes in XAML
- **Графики:** ScottPlot 5
- **Среда разработки:** Microsoft Visual Studio Community 2022


***На данный момент проект находится в разработке!!!***
