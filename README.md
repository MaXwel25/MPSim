# MPSim: приложение для анализа многофазной системы обработки данных

Программная модель для исследования работы конвейерного процессора (многофазной системы обработки данных). Приложение моделирует поток заданий, проходящих через `k` последовательных фаз с функциональными устройствами (ФУ) и буферами.
